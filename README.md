# Restful-booker Playwright Framework

## Tech Stack
- Playwright
- TypeScript
- GitHub Actions

### Positive Tests
- Get Booking
- Create Booking
- Update Booking
- Delete Booking

### Negative Tests
- Non-existent Booking
- Delete Booking Twice
- Update Deleted Booking

### Authorization Tests
- Update Without Authentication
- Delete Without Authentication

## Run Tests
npm install
npx playwright test
